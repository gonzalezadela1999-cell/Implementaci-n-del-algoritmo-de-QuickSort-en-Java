Desarrollar la implementación del algoritmo QuickSort en Java. El código debe incluir: 
- Una clase principal con un método main que pruebe el algoritmo con un arreglo de ejemplo.
- Un método quickSort que ordene el arreglo.
- Un método partition que realice la partición usando el último elemento como pivote.
- Un método auxiliar printArray para mostrar el arreglo antes y después de ordenar.
El archivo debe llamarse QuickSort.java.
